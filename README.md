# Barbershop Application

## Autori
- Cazan Cristian Madalin
- Caliman Vladut

## Descriere
Acest proiect reprezintă o aplicație de tip Barbershop, realizată în cadrul disciplinei Verificarea si  Validarea Sistemelor Software. Aplicația gestionează programările, serviciile și informațiile despre clienți. 
Arhitectura este organizată pe straturi: domain, entity, repository și service layer. Proiectul evidențiază principiile de validare și testare a componentelor software.

## Functionalități principale
- Gestionarea clienților
- Gestionarea frizerilor
- Administrarea programărilor
- Gestionarea serviciilor disponibile
- Validarea datelor introduse în sistem

## Structura proiectului
- **domain** – modele și logica domeniului
- **entity** – reprezentarea obiectelor persistente
- **repository** – acces la baza de date
- **service layer** – logica de business și validări
